Features

Everything apparel businesses need, nothing they don't.

Purpose-built modules that extend Business Central with industry-specific intelligence. No bloat, no workarounds.

Product Variants

Auto-generate thousands of color-size combinations with semantic codes. Visual matrix for instant inventory overview.

Seasonality

Organize collections into seasons with delivery groups. Customer-specific ship dates and cancellation windows built in.

Soft Allocations

Match demand to supply with priority sequencing. Identify shortages before they become problems.

Landed Costs

Automatic accruals at receipt. HTS code integration for duty rates. Accurate inventory costs from day one.

Product Attributes

30 custom attribute slots per item. Fabric, fit, gender, collection - structured data without schema changes.

Item Statuses

Lifecycle control with 7 blocking operations. Manage new product intro, discontinuation, and closeout workflows.

How It Works

Up and running in four steps.

A guided setup wizard configures everything. No consultants required for the basics.

01

Install & Configure

Install from AppSource. The 14-step assisted setup wizard guides you through dimensions, number series, and module activation.

02

Define Your Catalog

Set up colors with Pantone codes, size ranges, and variant templates. Auto-generate all combinations with one click.

03

Plan Your Seasons

Create seasons, delivery groups, and customer-specific timelines. Configure allocation priorities and landed cost rules.

04

Run Your Business

Variants flow through every document - sales, purchasing, inventory, warehouse. Everything works with native BC workflows.
